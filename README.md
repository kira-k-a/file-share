# Сервис обмена файлами
Простой сервис для обмена файлами с веб-интерфейсом
Backend:
Node.js + Express
SQLite
Multer
TypeScript
Frontend:
HTML/CSS/JavaScript

## Запуск
1. Клонируйте проект и перейдите в папку:
   ```bash
   git clone https://github.com/yourusername/file-share.git
   cd file-share/backend
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запуск проекта:
   ```bash
   npm run dev
   ```
   Откройте файл `file-share/frontend/index.html` в браузере (двойным кликом ЛКМ)

## Применение
1. Нажать кнопку "Выбрать файл", откроется диалоговое окно проводника, выбрать требуемый файл (двойным кликом ЛКМ)
	1а. Для примера создан файл "Privet.txt", находится в корне проекта `file-share/ `.
	1b. Файл загрузиться на сервер и под кнопкой "Выбрать файл" отобразиться имя выбранного файла
2. Нажать кнопку "Загрузить", под кнопрой отобразиться текст "Ссылка на скачивание файла "имя файла": ссылка"
	2а. В блоке "Статистика загруженных файлов" отобразиться информация о файле (имя, дата загрузки, дата скачивания, ссылка)
	
## Настройки хранения
Файлы сохраняются в папку uploads/
Метаданные в SQLite базе data/files.db
Автоочистка запускается каждые 7 дней
